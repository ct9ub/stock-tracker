# 주식 단타 관리 시스템 - 작업 이력

## 프로젝트 개요
- **목적**: 주식 단타 매매 보조 시스템 (인플루언서 추천 종목 추적 + 매매 일지)
- **기술 스택**: Python, Streamlit, SQLite, PyKRX, Plotly
- **실행 방법**: `streamlit run app.py`
