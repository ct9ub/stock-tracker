# 주식 단타 관리 시스템 - 작업 이력

## 프로젝트 개요
- **목적**: 주식 단타 매매 보조 시스템 (인플루언서 추천 종목 추적 + 매매 일지)
- **기술 스택**: Python, Streamlit, SQLite, 네이버 금융 크롤링, Plotly, APScheduler
- **실행 방법**: `streamlit run app.py`
- **GitHub**: https://github.com/ct9ub/stock-tracker

#### 기술적 결정사항
- 주가 데이터: 네이버 금융 크롤링 (무료, 계좌 불필요)
- 자동 업데이트: APScheduler BackgroundScheduler (1시간 간격, 평일 장중)
- 한국투자증권 API: 미연결 (인플루언서 추적에는 일별 종가면 충분)
- DB: SQLite (서버 불필요, 파일 1개로 관리)
- UI: Streamlit + Plotly (시각적 비교 분석)
